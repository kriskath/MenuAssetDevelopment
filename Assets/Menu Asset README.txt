Required Packages from the Unity Package Manager:
1. TextMeshPro v3.0.6 
2. Input System v1.1.1

Notes: 
1. Menu buttons may not work due to Old Input System being used. 
	To change this go to Edit>Preferences>Player>Other Settings>Configuration>Active Input Handling 
	then select Input System Package (New) or Both to get functionality of new input system
2. Confirmed Unity version asset works on is version 2020.3.15f2
